# OmenBot - NodeJS

This is a bot manager for omenium app, which will register/login some bots and manage them playing in omenium.
Also It will take action for each bot, in special occasions, such as bot running out of Chips, Gas or other tokens or stuff.
It's completely configurable, Such as operator being able to specify the amount of bot participation in leagues and periodical leagues, and so many other options.
